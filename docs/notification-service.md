# Order Status Notification Service

## Overview

The notification service automatically sends email notifications to customers whenever their order status changes. This keeps customers informed about their order progress in real-time.

## Status Types & Email Content

| Status | Emoji | Color | Message |
|--------|-------|-------|---------|
| `initiated` | рҹ“Ӣ | Gray | Order received, awaiting processing |
| `processing` | вҡҷпёҸ | Yellow | Team has started working on order |
| `in_progress` | рҹ”Ё | Blue | Order is actively being worked on |
| `completed` | вң… | Green | Order completed, deliverables ready |
| `cancelled` | вқҢ | Red | Order has been cancelled |

---

## Trigger Points

### 1. Admin Assigns Employee
```
assignEmployeeToOrderItem() вҶ’ sends "Processing" email
```

### 2. Employee Updates Status
```
updateOrderItemStatus() вҶ’ sends email for new status
```

### 3. Admin Updates Status Directly
```
updateOrderItemStatusAdmin() вҶ’ sends email for new status
```
